# **House Price Prediction**

## Overview

This is a practice project in Machine Learning focused on predicting house prices using a real-world dataset. The purpose of this project is to improve practical understanding of regression modeling, data preprocessing, and performance evaluation.

This project was completed as a personal exercise to gain hands-on experience with data handling, model training, and results interpretation.



## Dataset

The dataset used in this project contains various features related to housing attributes such as:
- Number of rooms
- Location indicators
- Area (square footage)
- Other numerical and categorical predictors

The goal is to understand how different features correlate with the target variable (house price).



## What This Project Demonstrates

This repository showcases real skills in:

### Data Processing
- Importing and cleaning data
- Handling missing values
- Encoding categorical variables
- Feature selection and transformation

### Exploratory Data Analysis (EDA)
- Summary statistics
- Visual inspection of feature relationships



## Machine Learning Model Used

This project trains and evaluates a regression model to predict house prices. The model selection, training, and validation demonstrate:
- XGBoost Regressor
- Model fitting
- Error calculation
- Prediction interpretation

Specific regression algorithms and performance metrics are used to understand how model quality can be assessed.
